# راهنمای نصب برنامه‌ریز من

## مرحله ۱ — نصب Flutter

1. برو به سایت: https://flutter.dev/docs/get-started/install/windows
2. فایل ZIP رو دانلود کن
3. محتویاتش رو در یه پوشه مثل `C:\flutter` extract کن
4. در ویندوز، Environment Variables رو باز کن:
   - روی "This PC" راست کلیک → Properties → Advanced system settings → Environment Variables
   - در "System variables" → "Path" رو انتخاب کن → Edit
   - آدرس `C:\flutter\bin` رو اضافه کن

## مرحله ۲ — نصب Android Studio

1. از سایت https://developer.android.com/studio دانلود کن
2. نصب کن (همه گزینه‌های پیش‌فرض رو قبول کن)
3. بعد از نصب Android Studio رو باز کن
4. در SDK Manager، نسخه‌های زیر رو نصب کن:
   - Android SDK Platform 34
   - Android SDK Build-Tools

## مرحله ۳ — تأیید نصب

یه Command Prompt باز کن و بنویس:
```
flutter doctor
```
اگه همه چیز سبز بود، آماده‌ای!

## مرحله ۴ — ساخت اسکلت اندروید (این مرحله حیاتیه، رد نکن!)

پوشه `android` که برات فرستادم فقط شامل تنظیمات دسترسی‌ها و اعلان‌هاست — فایل‌های اصلی پروژه اندروید (Gradle، MainActivity، آیکون‌ها) رو نداره، چون این‌ها رو Flutter خودش می‌سازه. اگه این مرحله رو انجام ندی، پروژه اصلاً بیلد نمی‌شه.

1. داخل پوشه `planner_app`، پوشه `android` رو **کامل پاک کن**
2. Command Prompt رو باز کن و برو به پوشه پروژه:
   ```
   cd C:\planner_app
   ```
3. این دستور رو بزن تا Flutter اسکلت اندروید رو از نو بسازه:
   ```
   flutter create .
   ```
4. حالا فایل `android\app\src\main\AndroidManifest.xml` تازه‌ساز رو باز کن (با Notepad یا هر ادیتور)
5. فایل `مجوزهای_اضافی.txt` که همراه پروژه فرستادم رو باز کن — خط‌هایی که توش هست (مجوزهای اعلان و alarm) رو داخل تگ `<manifest>` فایل تازه‌ساز کپی کن، درست قبل از تگ `<application`
6. همچنین دو تا `<receiver>` که تو همون فایل هست رو داخل تگ `<application>` (قبل از بسته شدنش با `</application>`) کپی کن
7. مقدار `android:label` رو هم می‌تونی به `"برنامه‌ریز من"` تغییر بدی

## مرحله ۵ — بیلد گرفتن از اپ

1. Command Prompt رو باز کن
2. برو به پوشه‌ای که این فایل‌ها رو گذاشتی:
   ```
   cd C:\path\to\planner_app
   ```
3. پکیج‌ها رو دانلود کن:
   ```
   flutter pub get
   ```
4. گوشیت رو به لپ‌تاپ وصل کن و USB Debugging رو فعال کن:
   - Settings → About Phone → روی "Build Number" ۷ بار ضربه بزن
   - Settings → Developer Options → USB Debugging رو فعال کن
5. اپ رو بیلد و نصب کن:
   ```
   flutter run --release
   ```
   یا اگه فقط می‌خوای فایل APK داشته باشی:
   ```
   flutter build apk --release
   ```
   فایل APK در مسیر `build\app\outputs\flutter-apk\app-release.apk` قرار می‌گیره

## مشکلات رایج

**خطای "flutter not found"**: مرحله ۱ رو دوباره چک کن و مطمئن شو Path درست تنظیم شده

**خطای "Android license"**: بنویس `flutter doctor --android-licenses` و همه رو Y بزن

**گوشی شناخته نمی‌شه**: درایور USB گوشیت رو نصب کن (از سایت Xiaomi)

**خطا موقع `flutter create .`**: اگه گفت پوشه android از قبل وجوددارد، مطمئن شو کامل پاکش کردی (نه فقط خالیش کردی — خود پوشه رو حذف کن)

**اعلان‌ها روی گوشی کار نمی‌کنه**: مطمئن شو وقتی اپ اولین بار باز شد، اجازه‌ی اعلان (notification) رو تأیید کردی — این دیالوگ فقط یک‌بار میاد
